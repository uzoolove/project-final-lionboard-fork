# 기능 정의서

## 1. 공통 규칙

- 로그인 필요 기능은 비로그인 상태에서 접근 시 로그인 화면으로 이동한다.
- 작성/수정/삭제 권한은 작성자 본인에게만 부여한다.
- 실패 시 사용자에게 사유를 안내한다(권한 없음, 입력 오류 등).

## 2. 기능 상세

| 기능ID | 기능명 | 우선 순위 | 입력 | 처리 | 출력 | 예외 |
| --- | --- | --- | --- | --- | --- | --- |
| FD&#8209;01 | 회원가입 | 1순위 | 이메일, 비밀번호, 비밀번호 확인 | 이메일 중복 확인 후 회원 생성 | 회원가입 완료 메시지, 로그인 화면 이동 | 중복 이메일, 비밀번호 규칙 위반 |
| FD-02 | 로그인 | 1순위 | 이메일, 비밀번호 | 인증 성공 시 세션/토큰 발급 | 로그인 성공, 마지막 방문 게시판으로 이동 | 계정 없음, 비밀번호 불일치 |
| FD-03 | 게시판 선택 | 1순위 | 게시판 탭(정보/자유/Q&A) | 선택된 게시판으로 목록 갱신 | 게시글 목록 | - |
| FD-04 | 게시글 목록 조회 | 1순위 | 페이지 번호 또는 스크롤 | 최신순 기본 정렬로 목록 제공 | 게시글 카드/리스트, 작성자, 작성일, 댓글 수 | 결과 없음 안내 문구 |
| FD-05 | 게시글 상세 조회 | 1순위 | 게시글 클릭 | 게시글 내용/댓글 로드 | 제목, 본문, 작성자, 작성일, 댓글 목록 | - |

| FD-07 | 게시글 수정 | 1순위 | 제목, 본문 | 작성자 권한 확인 후 수정 | 수정 완료, 상세 화면 이동 | 권한 없음 |
| FD-08 | 게시글 삭제 | 1순위 | 삭제 확인 | 작성자 권한 확인 후 삭제 | 목록 화면 이동 | 권한 없음 |
| FD-09 | 댓글 작성 | 2순위 | 댓글 내용 | 유효성 검사 후 등록 | 댓글 목록 갱신 | 빈 댓글 입력 |
| FD-10 | 댓글 삭제 | 2순위 | 삭제 확인 | 작성자 권한 확인 후 삭제 | 댓글 목록 갱신 | 권한 없음 |
| FD-11 | 검색 | 2순위 | 키워드(제목/본문) | 검색 조건으로 목록 조회 | 검색 결과 목록 | 결과 없음 안내 문구 |
| FD-12 | 페이지네이션 | 2순위 | 페이지 번호 또는 더보기 | 선택한 페이지로 목록 갱신 | 해당 페이지의 목록 | 결과 없음 안내 문구 |
